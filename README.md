# Equity Portfolio Analysis

Questa è una web app costruita con **Streamlit** per l'analisi di portafogli azionari. Utilizza `yfinance` per recuperare dati storici e visualizzare le performance.

## Installazione
1. Clona il repository: `git clone https://github.com/PaoSam/equity-portfolio.git`
2. Installa le dipendenze: `pip install -r requirements.txt`
3. Avvia l'app: `streamlit run App.py`
